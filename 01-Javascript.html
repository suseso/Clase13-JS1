Codigo JS incrustado en el HTML

Una manera de incluir código JavaScript en un documento HTML es utilizar la etiqueta script.

Si analizas el código de ésta página, encontraras el script dentro del head de la misma.

Para ver que hace ese código, debes usar la consola que provee el navegador web. Puedes acceder a ella, por ejemplo, presionando F12.


Puedes probar que ocurre si pones ese script en otra parte del documento HTML!
